# Calculadora_IA

Calculadora de depreciación Ross-Heidecke para valoración de inmuebles.

## Verla localmente

Puedes abrir la página directamente en tu navegador:

- Abre [calculadora.html](calculadora.html)
- O usa un servidor simple:

```bash
cd /workspaces/Calculadora_IA
python3 -m http.server 8000
```

Luego entra a:

```text
http://127.0.0.1:8000/
```

## Compartirla con otras personas

Para que otras personas puedan usarla, puedes publicar el proyecto en GitHub Pages o en cualquier servicio que sirva archivos estáticos.

### Opción recomendada: GitHub Pages
1. Sube este repositorio a GitHub.
2. En la configuración del repositorio, activa GitHub Pages.
3. Selecciona la rama principal como fuente.
4. La página quedará disponible en una URL pública como:

```text
https://<tu-usuario>.github.io/Calculadora_IA/
```

La página principal [index.html](index.html) redirige automáticamente a [calculadora.html](calculadora.html).
